# foodtruck-map

## 3주차 목표

발표가 있는 주

구매자 입장에서 구현하는 것 먼저. 판매자 메인도 구현하는 것이 좋을 듯.

## detail

Frontend

- 트럭 상세정보(구매자가 보는 페이지)
    - 푸드트럭 영업정보 확인하기
    - 메뉴 보기
- 맵
    - DB로부터 데이터를 받아 마커 찍기.
    - 마커를 클릭하면 상세페이지로 이동.
- 판매자 메인페이지 view
    - 내 푸드트럭 영업정보 수정하기
    - 메뉴 관리하기
- 검색, 리스트 뷰, 알림 등에 사용되는 리스트 폼 구현

Backend

- 판매자 메인페이지 crud
- 구매자 - 판매자의 페이지 리뷰, 댓글
